# /monitor — Configurable metric watchers with alerts

Read and follow the instructions in `skills/monitor/SKILL.md`.

Command: $ARGUMENTS

**Quick reference:**
- `/monitor create <name>` — Create a new monitor (interactive setup)
- `/monitor list` — Show all monitors with current status
- `/monitor check [name]` — Manually trigger a check (one or all due)
- `/monitor history <name> [limit]` — View recent check results
- `/monitor update <name> ...` — Modify a monitor's config
- `/monitor delete <name>` — Remove a monitor and its history
