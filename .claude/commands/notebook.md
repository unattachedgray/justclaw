# /notebook — Document-grounded analysis (NotebookLM-style)

Read and follow the instructions in `skills/notebook/SKILL.md`.

Command: $ARGUMENTS

**Quick reference:**
- `/notebook create <name> <path>` — Ingest a folder of documents
- `/notebook ask <name> <question>` — Query with source-grounded answers
- `/notebook overview <name>` — Generate notebook guide
- `/notebook faq <name>` — Generate FAQ from sources
- `/notebook sources <name>` — List indexed files
- `/notebook list` — List all notebooks
- `/notebook refresh <name>` — Re-ingest from same directory
- `/notebook delete <name>` — Remove a notebook
